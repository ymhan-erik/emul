# 에뮬레이터 보드 버전에 따른 구분
## 버전확인  
'/emualtor' 폴더에서 'diag'명령을 사용하면 버전을 확인 할 수 있다.  
cd /emulator  
./diag  

버전은 두 가지
Version 2.x
      FPGA Bitstream Date     : 20201013 
      FPGA Bitstream Version  : 10021000 (v2.1)
      
Version 1.x
      FPGA Bitstream Date     : 20201013 
      FPGA Bitstream Version  : 10011000  (v1.1)
